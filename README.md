# Zhonghe_Livability_Analysis

## 題目主題

利用政府資料的不動產實價登錄資訊，從價格、環境、交通便利性、生活機能等角度分析並總結出對普羅大眾而言，新北市中和區哪個地段是宜居地段。

## 分析目標及指標

預計將資料整理為

1. 土地區段位置建物區段門牌
2. 總價
3. 每坪單價
4. 從以下面向與指標進行分析：

* 價格可負擔性：分析房屋單價與總價，按地段繪製價格分佈圖。
* 生活機能：計算房屋周邊學校、公園、超市、捷運站等的密度和距離。
* 交通便利性：根據鄰近的捷運站、火車站、主要道路(高速公路)進行地段劃分，分析交通便利程度
* 環境品質：依據政府公佈的空氣品質、綠化比例數據進行綜合評分。
* 使用pandas套件整理資料，處理重複值、缺失值。
* 使用’Matplotlib’和’Seaborn’繪製以下圖表：
  - 房價熱力圖(顥示不同地段的單價分布)
  - 生活機能雷達圖(對比不同地段的便利程度)
  - 環境品質條形圖(按地段顥示空氣品質和綠化比例)
  - 使用地理可視化工具(如’folium’或’geopandas’)，呈現地段和房價，機能等多屬性圖層。
